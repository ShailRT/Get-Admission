Installation 
1. Install Python from - https://www.python.org/downloads/windows/ 
2. Clone this repo
3. cd into repo directory and run powershell
4. Type virtualenv . 
5. Run virtual environment by typing ./scripts/activate
6. Install dependencies by typing pip install -r requirements.txt
7. Run django server by typing python manage.py runserver 
8. Open link in your browser http://localhost:8000
9. Open VSCode and start making changes
